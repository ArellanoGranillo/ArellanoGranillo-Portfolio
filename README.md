# Personal Portfolio Website

**Functionality Screenshots Added**

This repository contains my personal portfolio website, designed to showcase my professional profile, skills, experience, and projects.

The website is structured as an online curriculum vitae and portfolio, combining clean design with clear content presentation.

---

## ✨ Features

- Personal profile and professional summary
- Skills overview
- Work experience
- Education and certifications
- Projects section
- Downloadable PDF certificates
- Responsive layout
- Clean, modern, editorial-inspired UI

---

## 🛠️ Technologies Used

- JSP (JavaServer Pages)
- HTML5
- CSS3
- Apache Tomcat 11
- Static assets (images and PDF files)

---
Functionality Screenshots Description
1. MainScreen: Displays the editorial-inspired landing page, featuring a professional summary and a clean, high-end visual aesthetic.

2. Projects: Shows the dynamic project gallery, showcasing a structured portfolio of development work with clear technical categorizations.

3. CV: Displays the integrated digital CV section, optimized for readability and featuring downloadable professional credentials.

4. ContactMe: Shows the dedicated contact interface, designed for professional inquiries with a focus on clean UX and responsive layout.
